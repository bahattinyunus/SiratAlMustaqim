# Sırat-ı Müstakim'e Katkıda Bulunma Rehberi
Bu projenin etik vizyonuna uygun katkılarınızı memnuniyetle karşılıyoruz.

## Nasıl Katkıda Bulunabilirsiniz?

1.  **Yeni Senaryolar:** Daha fazla etik ikileme ihtiyacımız var. Bunları `src/world.py` dosyasına ekleyin.
2.  **Mekanikler:** `src/morality.py` içindeki `MoralityEngine` (Ahlak Motoru) yapısını daha incelikli hale getirin (örneğin "Niyet" kontrolleri ekleyin).
3.  **Hata Düzeltmeleri:** Simülasyonun sorunsuz çalıştığından emin olun.

## Adab-ı Muaşeret (Davranış Kuralları)

*   **Haram İçerik Yok:** Açık bir olumsuz sonuç/ceza mekanizması olmadan etik dışı davranışları teşvik eden senaryolar eklemeyin.
*   **Saygı:** Yorumlar ve tartışmalar saygılı ve yapıcı olmalıdır.
*   **Temiz Kod:** PEP 8 standartlarına uyun.

## Pull Request (Çekme İsteği) Süreci

1.  Depoyu forklayın.
2.  Bir özellik dalı (feature branch) oluşturun (`git checkout -b ozellik/yeni-senaryo`).
3.  Değişikliklerinizi commit yapın.
4.  Dala push yapın.
5.  Bir Pull Request açın.

Hayırlı kodlamalar!
